# 🤖 Robotics Core UI

Nx моно-репозиторий: Angular + NestJS. Речевой синтез, распознавание голоса, балансировка нагрузки, базы данных.

## Технологии

- **Nx** — монорепозиторий
- **Angular** — фронтенд
- **NestJS** — бэкенд
- **TypeScript** — основной язык

## Возможности

- Синтез речи (TTS)
- Распознавание голоса (STT)
- Балансировка нагрузки
- Работа с базами данных

## Запуск

```bash
npm install
nx serve
```
